Vaya a Configuración > General > Acerca de, para averiguar el modelo de su dispositivo. El modelo de su dispositivo se puede encontrar a la derecha del Nombre del modelo.

Se admiten las versiones de software iOS/iPadOS 15.0 - 16.3 beta.
Para iOS 16, solo los dispositivos A11 son compatibles con iOS 16, por lo que son el iPhone 8, 8 Plus y X.
Si desea Jailbreak iOS 12.0 - 14, use checkra1n Jailbreak.

En lugar de ejecutar Semi-Tethered Jailbreak o Tethered Jailbreak, ejecute Remove Semi-Tethered Jailbreak o Remove Tethered-Jailbreak. En palera1n GUI v1.0 y v1.1, Remove Semi-Tethered no funciona. Así que usa Remove Tethered Jailbreak.

SI HACE JAILBREAK CON PALERA1N GUI V1.0, DEBE UTILIZAR PALERA1N GUI V1.0 PARA ELIMINAR EL JAILBREAK, ENTONCES PUEDE ACTUALIZAR A PALERA1N GUI V1.1 Y LUEGO RE-JAILBREAK SU DISPOSITIVO.

NO PUEDE REVERTIR/ELIMINAR UN JAILBREAK CON UNA VERSIÓN CON LA QUE NUNCA COMENZÓ EL JAILBREAK. ESTO ES PORQUE CUANDO EJECUTAS EL JAILBREAK, SE CREAN ARCHIVOS ESPECIALES PARA TU DISPOSITIVO Y ESTOS SE ALMACENAN EN LA CARPETA DONDE ESTABAS HACIENDO EL JAILBREAK PRIMERO.

Palera1n GUI requiere macOS Catalina o superior para ejecutarse, la compatibilidad con Linux llegará pronto. Y Windows nunca recibe soporte. En teoría, las versiones de macOS anteriores a Catalina deberían funcionar, pero el desarrollador probó su aplicación palera1n GUI solo en macOS Catalina, Big Sur y Monterey.

Cancele el proceso de jailbreak, luego vuelva a intentarlo, si reiniciarlo no lo soluciona, reinicie su dispositivo y computadora, luego vuelva a intentarlo. Si eso no lo soluciona, restaure rootfs y vuelva a intentarlo. Debería funcionar, eventualmente.

Todo lo que tiene que hacer es seguir intentando hacer jailbreak nuevamente, intentar reiniciar tanto el iPhone como la Mac e intentar usar una red Wi-Fi diferente. Eventualmente funcionará.

Abra una nueva ventana de terminal, luego escriba: sudo xattr -rd com.apple.quarantine *arrastre la carpeta GUI de palera1n aquí* Este comando soluciona cualquier problema potencial al cargar archivos que se descargaron de Internet.

Asegúrese de que la terminal tenga acceso completo al disco, así como sshd-ketgen-wrapper. Esto es solo para ejecutar cosas como ideviceinfo, idevicepair pair y más. Abra Preferencias del sistema -> Seguridad y privacidad -> Privacidad, luego presione el botón de bloqueo para realizar cambios, ingrese su contraseña, luego, una vez desbloqueada, busque acceso completo al disco, luego asegúrese de que la terminal y el sshd-ketgen-wrapper tengan acceso completo al disco.

Inicie el terminal, luego abra la carpeta palera1n en la carpeta GUI de palera1n, luego escriba terminal: python3 (arrastre el archivo palera1n.sh) -clean, luego presione enter.

Abra otra ventana de terminal, luego ejecute "git clone https://github.com/verygenericname/SSHRD_Script --recursive && cd SSHRD_Script", luego escriba "python3 sshrd.sh -clean"

Cuando ejecute los comandos anteriores en la terminal, ejecútelo sin comillas.
